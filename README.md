# 📊 Retail Sales Analytics Dashboard

## 📌 Project Overview

The Retail Sales Analytics Dashboard is an interactive business intelligence project built using **Python** and **Tableau Public**. The project analyzes retail sales data to identify trends, regional performance, and the impact of discounts on profitability.

Python was used for data cleaning and preprocessing, while Tableau was used to create an interactive dashboard that enables users to explore sales performance through dynamic visualizations and filters.

---

## 🚀 Project Objectives

- Analyze overall retail sales performance.
- Identify high-performing sales regions.
- Monitor monthly sales trends.
- Understand how discounts affect profit.
- Provide interactive filtering for better business insights.

## 📂 Dataset

The project uses the **Sample Superstore** retail dataset containing:

- Order Details
- Customer Information
- Product Information
- Sales
- Profit
- Discount
- Region
- State
- Segment

---

## 🧹 Data Cleaning (Python)

The dataset was cleaned and transformed using Python.

### Data Preparation Steps

- Removed duplicate records
- Handled missing values
- Converted date columns into datetime format
- Extracted:
  - Order Year
  - Order Month
  - Order Quarter
- Calculated:
  - Processing Days
  - Profit Margin (%)
- Exported cleaned dataset as **Superstore_Cleaned.csv**

### Visualizations

### Interactive Filters

- Order Year
- Region
- Category
- Segment

---

## 📈 Key Insights

- Sales peak during the end of the year, especially in November.
- The West region generates the highest sales.
- Higher discounts generally reduce profitability.
- Sales performance varies significantly across different states.
